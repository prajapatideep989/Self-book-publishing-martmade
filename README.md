Alpha Book - Self Book Publishing Website
Overview : 

Alpha Book is a self-book publishing web application where users can publish, buy, and sell books online.
Authors can upload their books for approval, and after admin approval, the books are displayed on the website for purchase.

This platform provides a simple marketplace for self-published authors while allowing the admin to manage book approvals and earn a commission from every sale.

Features :

User Features
User Registration and Login
Upload and Publish Books
Browse Available Books
Add Books to Cart
Purchase Books Online
View Book Purchase Status in Account

Admin Features :

Admin Login
Approve or Reject Submitted Books
Manage Published Books
Monitor Orders
Earn 10% commission on every book sale
Payment System
Buyer purchases a book
90% payment goes to the author
10% payment goes to the admin

Technologies Used :

Frontend: HTML, CSS, JavaScript
Backend: PHP
Database: MySQL
Server: XAMPP / Apache

Project Workflow :

User registers and logs in.
Author uploads book details.
Admin reviews the submitted book.
Approved books appear on the website.
Buyer purchases the book.
Payment is divided between admin and author.
